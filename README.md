<a name="english"></a>
## :gb: English

### Overview

A customizable, modern, and responsive website template for vacation rental properties. Features integrated booking, multi-language support, and SEO optimization.

### Features

- **Booking Integration** - Direct reservation system
- **Multi-Language** - i18n support
- **SEO Optimized** - Meta tags and structured data
- **Responsive Design** - Mobile-first approach
- **Photo Gallery** - Property showcase

### Quick Start

```bash
git clone https://github.com/fracabu/villa-azzurra.git
cd villa-azzurra
npm install
npm run dev
```
